## Axolotl Tools

Axolotl Tools is a Java-based command-line tool collection designed for file compression, decompression, and future extensions.

This project started as a simple ZIP compression/decompression tool and will continue to grow as I learn new concepts and techniques in Java. New features will be added as my skills improve.

### Design Notes

To keep the implementation simple and reliable, the current version processes one compression or decompression task at a time.  
Although some basic multithreading concepts have been learned, concurrency is intentionally not used yet.  
Due to academic commitments, the time available for learning and improving this project is limited, so development and feature additions may progress gradually.  
This may be revisited in future versions as understanding improves.


### Project Goals

- Serve as a long-term personal Java learning project
- Turn newly learned concepts into practical tool features
- Gradually extend and refactor the project over time
- Practice Java IO, file system operations, and tool-oriented design

### Current Features

- ZIP file compression (files and directories)
- ZIP file decompression
- Command-line menu interaction

### Planned & Future Features

- Additional file processing tools
- Performance and memory usage improvements
- Code refactoring and structural optimization
- New features inspired by future learning topics

### Disclaimer

This project is primarily for learning purposes.  
It is not intended for production use, and its performance and robustness are limited compared to mature tools.

---

This project reflects my learning progress and will evolve as my understanding of Java deepens.

## Axolotl 工具

Axolotl 工具是一个基于 Java 的命令行工具集，专为文件压缩、解压缩以及未来的扩展而设计。

本项目最初是一个简单的 ZIP 压缩/解压缩工具，随着我学习 Java 的新概念和新技术，它将不断发展壮大。随着我技能的提升，新功能也会陆续添加。

### 设计说明

为了保持实现的简洁性和可靠性，当前版本一次只处理一个压缩或解压缩任务。

虽然已经掌握了基本的多线程概念，但目前并不打算使用并发。

由于学业繁忙，我用于学习和改进本项目的时间有限，因此开发和功能添加可能会循序渐进。

随着理解的加深，未来版本可能会重新考虑并发。

### 项目目标

- 作为一项长期的个人 Java 学习项目

- 将新学到的概念转化为实用的工具功能

- 随着时间的推移逐步扩展和重构项目

- 练习 Java IO、文件系统操作和面向工具的设计

### 当前功能

- ZIP 文件压缩（文件和目录）

- ZIP 文件解压缩

- 命令行菜单交互

### 计划及未来功能

- 额外的文件处理工具

- 性能和内存使用方面的改进

- 代码重构和结构优化

- 受未来学习主题启发的新功能

### 免责声明

本项目主要用于学习目的。

它不适用于生产环境，并且与成熟的工具相比，其性能和稳定性有限。
